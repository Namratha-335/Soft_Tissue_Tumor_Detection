## Deploy to Streamlit Cloud
1. Push this project to GitHub.
2. Go to https://share.streamlit.io
3. Click “New app” and select your GitHub repo.
4. Set file path to `app.py`.
5. Deploy.

## Note
- The app uses the file `tumor_transfer_model.h5`.  
- If the file is large, host it externally and put the URL in Streamlit Secrets.
